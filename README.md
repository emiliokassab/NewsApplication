News app:
A simple android application to display news based on an APIKEY from Gnews

Features:
Fetch N articles
Search by keyword
Filter articles
Click to Read
Image Display

Technologies Used:
Language: Java
IDE: Android Studio previously installed on my laptop
API:Gnews API
Libraries:
Retrofit 2.9.0: Rest API communication
Gson - JSON parsing
Glide - Image loading
RecyclerView - Display article lists 
CardView - Modern card UI design

Tested the project on an pixel 8 API 33 virtual emulator on android studio

How to Run:

1-Clone the repository
2-Open in android studio
3-Add your personal API KEY
4-Build and run
